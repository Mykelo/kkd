# Pliki:
-dynhuf.py - rozwiązanie zadania na ocenę 4

# Uruchomienie:
Plik dynhuf.py należy uruchomić poleceniem python3. Program przyjmuje 3 parametry:
    python3 entropy.py --encode [nazwa pliu kodowanego] [nazwa pliku wyjściowego]
lub analogiczne polecenie dekodujące:
    python3 entropy.py --decode [nazwa pliu zakodowanego] [nazwa pliku wyjściowego]
